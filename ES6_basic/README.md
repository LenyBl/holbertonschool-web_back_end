# ES6 Basic

## Overview
Introduction to ES6 (ECMAScript 2015) fundamentals and modern JavaScript syntax.

## Topics Covered
- `const` and `let` declarations
- Arrow functions
- Template literals
- Destructuring
- Classes
- Modules (import/export)
- Promises
- Default parameters
- Rest/spread operators

## Learning Outcomes
Understand core ES6 features and their practical applications in modern JavaScript development.
