# ES6 Data Manipulation

Small collection of exercises to practice ES6 features for data handling (map, filter, reduce, spread, destructuring, etc.).

## Requirements
- Node.js (v14+ recommended)

## Install
```bash
npm install
```

## Usage
- Run a script:
```bash
node path/to/script.js
```
- Run tests (if present):
```bash
npm test
```

## Structure
- `/*.js` — exercise files
- `test/` — tests (when included)
