# full_server

Minimal Node.js server for learning and testing REST endpoints.

## Prerequisites
- Node.js (v18+)
- npm

## Setup
```bash
cd full_server
npm install
```

## Run
```bash
# default: PORT=3000
npm start
# or
node src/index.js
```

## Usage
Open http://localhost:3000 or run:
```bash
curl http://localhost:3000
```

See source code for available routes and behavior.
