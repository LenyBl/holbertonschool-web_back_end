# ES6 Promises

Short exercises focused on using **JavaScript Promises** in Node.js.

## Learning goals
- Create and return Promises
- Handle resolve/reject states
- Chain Promises with `.then()` and `.catch()`
- Use `Promise.all`, `Promise.race`, and error handling patterns

## Requirements
- Ubuntu 24.04.3 LTS
- Node.js / npm
- ESLint (if required by project)

## Run
```bash
npm install
npm test
```
